Profiler API

An API endpoint to get list of users.

How to use

You can use AJAX to call the Profiler API and will receive a collection of users in return.

http://{{host}}/api/v1/profiles

API Errors

If there is error on the endpoint a sample error response will be receive.

{"error": "oops, something has gone wrong."}

Using filter

Requesting users with limit filter

http://{{host}}/api/v1/profiles?limit=5

Filtering by gender i.e male,female,other

http://{{host}}/api/v1/profiles?gender=female

Filtering by nationality i.e US,TR,FR,CH,DE,CA

http://{{host}}/api/v1/profiles?nationality=US

Filtering by title Mr,Mrs,Miss

http://{{host}}/api/v1/profiles?title=Mr

Using multiple filters at once

http://{{host}}/api/v1/profiles?title=Mr&nationality=US&gender=female&limit=100

Pagination

You can request multiple pages of a seed with the page parameter.

http://{{host}}/api/v1/profiles?title=Mr&nationality=US&gender=female

You can also request use filters with pagination.

http://{{host}}/api/v1/profiles?title=Mr&nationality=US&gender=female&per_page=100&page=1
